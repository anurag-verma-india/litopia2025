# 🚀 Litopia 2025 landing Page

## 🛠️ Installation

1. Clone the repository:

```bash
git clone https://github.com/anurag-verma-india/litopia2025.git
```

2. Install dependencies:

```bash
cd litopia2025
npm install
```

## 🚀 Usage

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
